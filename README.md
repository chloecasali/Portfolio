
# Interactive Personal Portfolio Website

This is a code bundle for Interactive Personal Portfolio Website.
The original project is available at:
https://www.figma.com/design/JqObwkkWS8BL2Ls1aPJg8y/Interactive-Personal-Portfolio-Website.

## Run Locally (without Docker)

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Run with Docker (Development, hot reload)

```bash
docker compose up --build portfolio-dev
```

Then open `http://localhost:5173`.

## Run with Docker (Production build)

```bash
docker compose up --build portfolio-prod
```

Then open `http://localhost:8080`.
